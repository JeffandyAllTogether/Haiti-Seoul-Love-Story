## Installation

### Prerequisites
- Python 3.12 or higher
- pip

### Local Setup

1. Clone the repository
```bash
git clone https://github.com/YOUR_USERNAME/haiti-seoul-love-story.git
cd haiti-seoul-love-story
```

2. Install dependencies
```bash
pip install streamlit
```

3. Run the app
```bash
streamlit run haiti_seoul_streamlit.py
```

4. Open your browser to `http://localhost:8501`
